# Chapmans AutoCentre

This is a simple React website for **Chapmans AutoCentre**, showing available cars with links to AutoTrader and a contact section.

## Features

- Display car listings with images, price, mileage, and fuel type
- Click on a car to view its listing on AutoTrader
- Contact section with phone, address, email, and WhatsApp link
- Fully responsive and mobile-friendly layout
- Clean, modern design

## Live Demo

[View Live Site](https://your-vercel-link.vercel.app)

## Technologies Used

- React
- HTML & CSS (inline styles)
- Hosted images via Postimages

## How to Run Locally

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server
